### Actual features:
+ List of Job offers.
+ Job Detail.
+ Share Job Offer.

### Future features:
+ Job Filter.
+ Save Job Offers.
+ JVM tests
+ UI tests using Espresso
   
### App Architecture:
+ Model View Presenter (MVP)
+ Package by feature
+ Kotlin
+ Rxjava
+ Dependency Injection (Dagger 2)
