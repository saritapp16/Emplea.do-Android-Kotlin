+ Search by Category.

### Future features:
+ Integrate the app With a new Backend.
+ Dependency Injection.
+ JVM tests
+ UI tests using Espresso
   
### App Architecture:
+ Model View Presenter (MVP)
+ Package by feature
+ Kotlin
+ Rxjava
